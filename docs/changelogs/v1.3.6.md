# v1.3.6 (2026-06-05)

A patch release with a CI workflow fix and documentation updates.

## Features and Improvements

_No notable features in this patch release._

## Development, Testing, and CI/CD

* **ci(release): repair orphaned draft tag_name on retag**: When a git tag is deleted and re-created between draft release creation and merge, GitHub orphans the draft release — setting `tag_name` to `untagged-<hash>` while preserving `name`. The release-finalize workflow now falls back to matching by `name` when the primary `tag_name` lookup misses, repairs the `tag_name` to point at the real tag, and then publishes correctly, preventing `Draft release for <tag> not found` failures. ([**@myasnikovdaniil**](https://github.com/myasnikovdaniil) in #2761, backport #2830).

## Documentation

* **[website] docs: exclude loop devices from LVM global_filter**: Adds `/dev/loop` to the documented LVM `global_filter` in the Talos install guides (v0, v1.4, and next) so the host does not scan loop-mounted disk images during installation ([**@kvaps**](https://github.com/kvaps) in cozystack/website#563).
* **[website] docs(networking): Cilium Gateway API — architecture, security, migration**: Adds a new `networking/gateway-api.md` page describing the Cilium-backed Gateway API feature — platform-level and per-tenant toggles, managed certificate support via cert-manager, and migration guidance from the legacy Ingress approach ([**@lexfrei**](https://github.com/lexfrei) in cozystack/website#509).
* **[website] docs(networking): publish Kubernetes API endpoint via external-dns with kuberture**: Adds documentation for the `kuberture` system package, which publishes the control-plane endpoint via `external-dns` as a workaround for `external-dns` lacking `EndpointSlice` source support ([**@lexfrei**](https://github.com/lexfrei) in cozystack/website#539).
* **[website] docs(v0): pin cozystack download URLs in v0 docs**: Pins legacy v0 install documentation download URLs to specific release versions, fixing broken links caused by the installer manifest restructuring introduced in v1.0 ([**@lexfrei**](https://github.com/lexfrei) in cozystack/website#548).
