# v1.2.4 (2026-05-01)

Patch release with security and bug fixes.

## Fixes

* **fix(platform): migrate ACME HTTP-01 to ingressClassName API**: Update ACME HTTP-01 handling to use the ingressClassName API, improving compatibility with newer ingress controllers and preventing ACME challenge regressions. ([**@myasnikovdaniil**](https://github.com/myasnikovdaniil) in #2438)

* **fix(api): prevent IDOR in TenantNamespace Get and Watch handlers**: Tighten access checks in TenantNamespace Get and Watch handlers to prevent insecure direct object references (IDOR). This prevents tenants from observing or retrieving namespaces they shouldn't have access to. (Original PR: [**@IvanHunters**](https://github.com/IvanHunters) in #2471; backport: #2523)

## Security

* **fix(talos): bump Talos to v1.12.7 (CVE-2026-31431)**: Upgrades bundled Talos to v1.12.7 to address CVE-2026-31431. Users should upgrade to this release to receive the security fix. ([**@kvaps**](https://github.com/kvaps) in #2545; backport: #2548)

## Documentation

No website documentation changes were detected in the release window.
