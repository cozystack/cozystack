# v1.2.4

Patch release containing bug fixes and backports.

## Fixes and improvements

* **fix(platform)**: migrate ACME HTTP-01 to ingressClassName API — improves ACME handling for ingress controllers ([**@myasnikovdaniil**](https://github.com/myasnikovdaniil) in #2438).
* **fix(kubernetes)**: set explicit ephemeral-storage on virt-launcher pods (see commit c611aa3...)
* **ci/build**: filter git describe to match only v* tags and replace GH_PAT with cozystack-ci GitHub App token (see commits a8b11ee..., 8b1f299...)
* **linstor**: update piraeus-server to v1.33.2 and related backports (see commits 69ccaa7..., 198c57c...)
* **monitoring**: restore infra dashboards in default variant (see commit 3b2d6a6...)
